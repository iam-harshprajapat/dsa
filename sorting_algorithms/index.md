## Notes

* Iterative implementations avoid recursive function calls and stack overhead.
* Recursive implementations are often easier to understand and closer to textbook definitions.
* Non-comparison sorting algorithms can achieve linear time complexity under specific constraints.
* Hybrid algorithms combine multiple sorting techniques to improve real-world performance.
